# Character Glitch

![Character Glitch preview](./social-preview.png)

Character-only browser experiments built as standalone HTML files.

Live site: https://willbearfruits.github.io/characterglitch/

Included pieces:

- `zalgo_vortex.html`
- `dejong_organism.html`
- `mycelium_network.html`
- `physarum_zalgo.html`
- `fractal.html`
- `charsort.html`
- `fibonacci_sphere.html`

Open any file directly in a browser, or use the index at `index.html`.
